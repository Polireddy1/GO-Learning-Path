Common commands

docker images  - lists all images available images on local m/c
Docker container ps   -    lists all running containers

Docker ps                       - running containers
Docker pull <image-name>                             -pulls the image to local m/c(download)
docker run <image-name>                                - download and run
Docker exec <container-id>
Docker stop<container-id>
Docker rm<container-id>                 -removing containers
dockerrmi<container-id>                   -removing images
